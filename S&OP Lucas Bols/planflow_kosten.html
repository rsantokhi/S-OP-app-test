Vergelijk dit met Streamline of Pigment
Streamline kost €2.000–€8.000 per jaar aan licenties (plus per-user kosten), zonder Oracle Fusion integratie en zonder multi-user web toegang. Pigment kost €30.000–€60.000 per jaar. PlanFlow op AWS kost €452/maand = €5.424/jaar voor de volledige productiestack — en jij bezit de applicatie volledig. De enige lopende kosten zijn de AWS infrastructuur.
02
Kritieke keuze: Oracle DB of PostgreSQL op AWS?
De database op AWS is niet hetzelfde als Oracle Fusion — Oracle Fusion blijft altijd de ERP-backend
✓ PostgreSQL op AWS RDS
Aanbevolen voor PlanFlow data
€185/maand
db.m6g.large · Multi-AZ · 100GB · eu-west-1
Open source, geen licentiekosten, volledig compatibel met onze Node.js stack. Bewezen schaalbaar tot miljoenen rijen. Dit is de slimme keuze voor PlanFlow's eigen data.
⚠ Oracle DB op AWS RDS
Alleen als klant dit vereist
€700–€900/maand
db.m5.large · License Included · Single-AZ · eu-west-1
4–5× duurder dan PostgreSQL voor hetzelfde werkprofiel. Alleen kiezen als de klant Oracle BYOL heeft én dit contractueel vereist is. Vermijd tenzij er een harde reden is.
Hoe de twee Oracle systemen zich tot elkaar verhouden
🔴
Oracle Fusion Cloud — de ERP van de klant. Hier halen we items, voorraad en PO's op via de REST API. Dit beheren wij NIET — dat is de klant's systeem. Kosten: €0 voor ons (klant betaalt licentie).
🐘
PostgreSQL op AWS RDS — onze eigen applicatiedatabase. Hier slaan we forecasts, PO workflows, gebruikers en audit logs op. Dit beheren wij wel. Kosten: €185/maand (Multi-AZ productie).
✓
Deze twee systemen werken naast elkaar — ze vervangen elkaar niet. PlanFlow is de planning laag. Oracle Fusion is de transactionele ERP laag.

Hoe je de kosten verder verlaagt
AWS biedt meerdere manieren om 20–50% te besparen zodra het gebruik stabiel is
AWS Savings Plans
30–50%
1-jaar commitment op Fargate compute. Vanaf €300/maand stabiel gebruik de moeite waard.
RDS Reserved Instance
30–40%
1-jaar reservering op de RDS instance. Op productie snel terugverdiend.
Fargate Spot
tot 70%
Voor batch jobs (forecast berekeningen) — niet voor de live applicatie zelf.

Advies: start met Starter, schakel na eerste klant naar Productie
1️⃣
Maand 1–2: Starter scenario (€119/maand). Server optuigen, Oracle Fusion koppelen, Azure AD testen. Geen risico, minimale kosten.
2️⃣
Maand 3 (go-live): Upgrade naar Productie (€452/maand). Multi-AZ database aan, blue-green deployment actief, eerste echte gebruikers.
3️⃣
Maand 12: AWS Savings Plan activeren. Kosten dalen naar ~€310/maand zonder iets aan de architectuur te veranderen.
4️⃣
Bij tweede klant: Enterprise scenario overwegen óf multi-tenant aanpak in dezelfde stack — afhankelijk van isolatievereisten van de klant.
